

<h3 align="center">Projetos em Python</h3>

## Sobre

Repositorio com scripts e automacoes em Python para WhatsApp, Instagram e utilitarios.

## Projetos

| Projeto | Descricao |
| --- | --- |
| `busca_automatica_google/` | Automacao de buscas no Google. |
| `instagrambot/` | Bots e automacoes para Instagram. |
| `whatsappbot/` | Scripts de envio automatizado no WhatsApp. |
| `wppmessage/` | Aplicacao com interface para agendamento de mensagens no WhatsApp. |
| `decifra_base64.py` | Script utilitario para decodificacao Base64. |

## Pre-requisitos

- Python 3.9+
- Google Chrome e/ou Firefox (dependendo do bot)

## Instalacao

1. Clone o repositorio.
2. Entre na pasta do projeto desejado.
3. Instale as dependencias descritas no `README.md` local.

## Execucao

1. Acesse a pasta do projeto:
```bash
cd nome-do-projeto
```
2. Execute o script principal conforme o projeto:
```bash
python3 arquivo.py
```

## Observacoes

- Os projetos baseados em Selenium foram atualizados para Selenium 4.
- Bots podem exigir ajustes periodicos por mudancas nas interfaces do WhatsApp/Instagram.
