<h3 align="center">Busca Automatica Google</h3>

## Sobre

Bot em Python que automatiza buscas no Google com palavras-chave predefinidas.

## Script

| Arquivo | Objetivo |
| --- | --- |
| `busca_automatica_google.py` | Executa buscas automatizadas e navegacao conforme regras do script. |

## Pre-requisitos

- Python 3.9+
- Google Chrome instalado

## Instalacao

```bash
pip install selenium webdriver-manager
```

## Execucao

1. Entre na pasta do projeto:
```bash
cd busca_automatica_google
```
2. Execute o script principal:
```bash
python3 busca_automatica_google.py
```

## Observacoes

- Ajuste as palavras-chave e parametros diretamente no script antes de executar.
- Mudancas na interface do Google podem exigir atualizacao de seletores Selenium.
